# 第一個可操作切片說明

## 目的

本原型展示宿舍報修流程的第一個可操作切片：

1. 宿舍學生提交報修單
2. 宿舍管理員查看待處理清單並派工
3. 維修人員查看已派工任務並標記完成

## 操作方式

1. 以瀏覽器開啟 `index.html`
2. 在「宿舍學生」頁面填寫報修資訊後送出
3. 切換到「宿舍管理員」頁面，將案件派給維修人員
4. 切換到「維修人員」頁面，點選「標記完成」

## 技術限制

- 使用 HTML、CSS、JavaScript
- 不使用正式登入
- 不使用正式資料庫
- 資料儲存在瀏覽器 `localStorage`

## 驗收條件對照

- FR-01：學生可送出報修單
- FR-02：送出後可看到「已提交」狀態
- FR-03：管理員可派工
- FR-04：維修人員可完成回報
- NFR-01：必填欄位空白時不能送出
